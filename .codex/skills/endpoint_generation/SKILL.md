# Skill: Minimal API Endpoint Generation

Use this skill when adding or modifying ASP.NET Core endpoints.

## Procedure

1. Read route group conventions and nearby endpoints.
2. Confirm request/response contract.
3. Place business logic in Application, not endpoint handlers.
4. Validate input.
5. Apply auth policies/admin gating.
6. Return consistent error responses.
7. Add contract/integration tests.
8. Update docs/OpenAPI notes if needed.

## Alpha bias

For Alpha, prefer stable authoritative endpoints over broad endpoint surface.
