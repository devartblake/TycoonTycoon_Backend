# Alpha Status Review Prompt

Review the current heartbeat state and produce an Alpha/Beta readiness report.

Read:
- `.codex/heartbeat/alpha-status.md`
- `.codex/heartbeat/current-blockers.md`
- `.codex/heartbeat/verification-log.md`
- `.codex/heartbeat/deferred-post-alpha.md`

Output:
- Current release readiness.
- P0 blockers.
- P1 risks.
- Verification failures.
- What to do next.
- What to defer.
- What needs user decision.
