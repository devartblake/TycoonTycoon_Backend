# Command: Add Contract Tests

Use this command when API contracts affect Flutter, sidecars, admin dashboard, or other clients.

## Instructions

1. Identify consumer.
2. Identify endpoint/contract.
3. Add success-path test.
4. Add important failure-path tests.
5. Add auth/authorization test where relevant.
6. Add idempotency or feature-flag test where relevant.
7. Run target tests.
