# Command: Review Security Boundary

Use this command for auth, KMS, secrets, economy/reward, admin, or sidecar security reviews.

## Instructions

1. Identify asset being protected.
2. Identify actors and trust boundary.
3. Identify data flow.
4. Identify possible abuse or leakage.
5. Check logs for secret exposure.
6. Recommend controls.
7. Classify as P0/P1/P2/P3.
