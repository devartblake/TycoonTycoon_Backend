What to verify

Use this surface to validate operator upload preparation before frontend clients consume the same contract.

  • The object key path matches the expected media category naming strategy.
  • Content type and size constraints are accepted by the backend.
  • The selected file is uploaded to the configured MinIO bucket or API fallback path.
  • The presigned URL expires within the backend-defined window.
